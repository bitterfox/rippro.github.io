概要(クリックして開閉)

大会目的

  • 夏休みのせいでプログラミングを忘れることをないようにする
  • ICPCの問題に少しでも慣れる
  • 実力の向上

大会日時

  • 8月7日(木) 20:00:00 ～ 9月28日(日) 23:59:59

表彰

  • 9月4日(木)の活動にて。上位入賞者には景品有。

内容

  • Aizu Online Judgeの問題をひたすら解き続ける。
  • 問題ごとに大会運営者が点数を振り、その合計点数で競い合う。

参加資格

  • RiPPro部員またはRiPProのOBであること。

参加費用

  • 回生×500円
  • 上位入賞者の超豪華景品の購入に利用します。

参加方法

  1. AOJへアクセス。
  2. USER MENUからRegisterを選択。
  3. 「User ID」「Password」「Name」を各自で入力。
  4. 「Affiliation」には「Ritsumeikan University」と入力。
  5. 「E-mail」は入力してもしなくても構わない。
  6. 「Send」で送信。
  7. SkypeなどでIDを伝える。

特別ルール

  • 何を見てもOK，考え方を知る目的で他人の解答を見るのもOK。
    • ただし、丸写しは禁ずる。
  • 得点計算において、回生に応じた重み付けを行う。
  • 上位入賞者3名には景品を用意。

順位

  • ポイントの高い人が上位となる。
  • 同着の場合は、下回生が上位となる。
  • AOJへの登録時から解いた問題の総ポイントが高いほうが上位となる。

ポイント計算ルール

  • 提出期間
    • コンテスト開始日から終了日までの間に正解した問題のポイントの合計が最終ポイントとなる。
  • 基本
    • ボーナス問題以外のAOJ上の問題は、どれを解いても1ポイント。
    • ただし、10000番台の問題は得点にはならない。
  • ボーナス問題
    • 通常の問題よりも配点が高く設定しているので、是非解いてもらいたい。
    • 配点が高いのは、解くことを推奨する問題だから。
    • ボーナス問題は、ICPC国内予選のA～Gとパソコン甲子園の何問か。
    • ICPCの問題が難しいと感じる人は、パソコン甲子園のボーナス問題から解くのがオススメ。
    • ボーナス問題は下表を参照。
  • 回生補正
    • 通常の方法でポイントを換算した後、回生で割る。
  • ボーナス問題の選出について
    • ICPC
    • JOI
    • PCK
    • あり本に登場するアルゴリズムがそのまま使える問題リストを作りたい。
    • TopCoder、Codeforces、AtCoderで得た得点をポイント換算したい。
