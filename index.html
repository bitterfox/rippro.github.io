進捗どうですか

これは何

  • 部内AOJ進捗状況を API によって取得し，可視化します
  • 表のヘッダーをクリックするとその列をキーとしてソートします
  • Volume 100 に含まれる問題はカウントされません
  • 自分がいなければ Github で追加してください
    • Github アカウントでログインし， 'Edit' ボタンをクリックすると編集できるようになります
    • 'memberIDs' がユーザーのIDを管理するコンマ区切りの配列です．ここに追加してください．
    • 'Commit changes' の下の欄にどういう変更をしたかを入力し，'Commit Changes' をクリックすると反映されます
  • このページは次のもの利用して作られています
    • Aizu Online Judge Application Program Interface - ユーザー情報・問題リストの取得
    • Github Pages - バージョン管理とページのパブリッシュ
    • jQuery - API呼び出し，DOM操作など
    • Flot - グラフ描画
    • tablesorter - 表のソート
